### รูปแบบคอลัมน์ที่สอง (ให้ตรงกับวิธีเขียนใน Word)

ทุกแถวใช้โครงเดียวกัน 4 บรรทัด (คัดลอกไปใส่ Comment / แผ่นงานได้ทันที):

1. **นำทางใน Word:** มุมมอง → แสดง → **แผ่นนำทาง** (หรือกด **Ctrl+F** ค้นหาข้อความในวงเล็บหลัง “ค้นหา:”)  
2. **องค์ประกอบที่เลือก:** ชนิดในเล่ม (เช่น ย่อหน้า Body / หัว Heading / แถว–คอลัมน์ในตาราง / รูป)  
3. **การดำเนินการ:** สิ่งที่ต้องทำใน Word (พิมพ์ / แทรกแถว / แทนที่รูป / อัปเดตฟิลด์)  
4. **ซิงก์อ้างอิง:** ไฟล์หรือเอกสาร repo ที่ใช้เทียบ

*(บรรทัดคั่นใน Markdown ใช้ `<br>` — เวลาคัดลอกไป Word อาจต้องลบ `<br>` หรือแปลงเป็นขึ้นบรรทัดใหม่ในเซลล์)*

## Document control / Acronym / Table of contents

| หัวข้อในเล่ม | คำสั่งแก้ใน Microsoft Word |
|--------------|------------------------------|
| Document Story / ตารางควบคุม | **1) นำทางใน Word:** Ctrl+F **ค้นหา:** `Document Name` หรือ `Version` ในตารางควบคุม หรือเลื่อนหาตารางแถวแรกมีหัว **Document Name**<br>**2) องค์ประกอบที่เลือก:** **ตาราง** (Table) ควบคุมเอกสาร — แถวหัวตารางและแถวข้อมูลใต้หัว **Document Name / Version / Status / Date / Responsible**<br>**3) การดำเนินการ:** คลิกใน **เซลล์** แต่ละช่องที่เกี่ยวข้อง → พิมพ์ค่า Version, Status, วันที่, ผู้รับผิดชอบ; ถ้ามีเซลล์หมายเหตุให้พิมพ์ baseline ลูกค้า (ชื่อชุดไฟล์ 4 ไฟล์)<br>**4) ซิงก์อ้างอิง:** รอบ review โครงการ + แหล่งลูกค้าใน `from customer/` |
| Acronym / san, chin, … | **1) นำทางใน Word:** Ctrl+F **ค้นหา:** `Acronym` หรือ `san =`<br>**2) องค์ประกอบที่เลือก:** ย่อหน้า/รายการคำย่อทีม<br>**3) การดำเนินการ:** แก้ข้อความให้ตรงทีมปัจจุบัน<br>**4) ซิงก์อ้างอิง:** ไม่บังคับลูกค้า |
| Table of content | **1) นำทางใน Word:** Ctrl+F **ค้นหา:** `Table of content` หรือ `สารบัญ`<br>**2) องค์ประกอบที่เลือก:** ฟิลด์ **สารบัญ** (TOC) ที่ Word สร้างไว้<br>**3) การดำเนินการ:** คลิกขวาที่สารบัญ → **อัปเดตฟิลด์** → เลือก **อัปเดตทั้งหมด** (หลังแก้หัวข้อและเลขหน้าเปลี่ยน)<br>**4) ซิงก์อ้างอิง:** โครงสารบัญต้องตรงหัว Heading ในเล่ม |
